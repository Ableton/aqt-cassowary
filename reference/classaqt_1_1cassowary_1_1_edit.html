Detailed Description

An edit constraint allows to suggest values for a variable.

It can be used to represent the user's wish.

Example
Solver {
Variable {
id: vX
Stay {}
Edit {
suggested: mouseArea.mouseX
when: mouseArea.pressed
}
}
}
MouseArea { id: mouseArea }
