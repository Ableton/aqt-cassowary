Detailed Description

A linear constraint.

A linear constraint allows us to set equations or inequalities that relate variables in the system. The solver will log errors in the presence of non-linear constraints or conflicting constraints with Required strength.

Example
Solver {
Variable { id: vX }
Variable { id: vY }
Constraint {
expr: eq(vX, times(2, vY))
}
Constraint {
expr: geq(plus(vX, vY), 42)
when: root.height > 100
}
// Error, non-linear expression
Constraint { expr: geq(times(vX, vX), vY) }
}
